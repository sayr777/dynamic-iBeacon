# Эксперименты

Этот раздел содержит отдельные исследовательские заметки и альтернативные направления развития проекта,
которые пока не входят в основную производственную реализацию.

- [Переход на Zephyr / nRF Connect SDK](zephyr-ncs-evaluation.md) — оценка сложности, ожидаемые плюсы и минусы, план минимального эксперимента и ссылки на официальную документацию.
- [Первичная настройка JDY-23](jdy23-initial-setup.md) — архивная инструкция по старой ветке решения на `JDY-23`, сохранена как неосновной экспериментальный материал.
- [BOM для варианта с солнечной батареей](solar-power-bom.md) — экспериментальный BOM и оценка стоимости для варианта `LiPo + солнечная панель`.
- [Альтернативный модуль E73-2G4M04S1A](e73-nrf52810-alternative.md) — архивная карточка альтернативы на `nRF52810`, не являющейся основным решением проекта.
