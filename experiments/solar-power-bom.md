# BOM для варианта с солнечной батареей

## Статус

Этот документ сохранён как **экспериментальный**.  
Вариант с `LiPo + солнечная панель` не является текущим основным решением проекта и вынесен из основного BOM.

## Идея

Вместо первичной батареи используется связка:

- солнечная панель;
- зарядный контроллер;
- LiPo-аккумулятор как буфер;
- LDO для питания BLE-модуля.

Такой подход интересен как исследовательское направление, но проигрывает основному варианту по предсказуемости и простоте производства.

## Комментарий по схемотехнике питания

```text
Солнечная панель 30×40 мм
    │
   [D1: BAT54] ── блок обратного тока
    │
   [CN3791: LiPo MPPT зарядник]
    │
   [LiPo 100 мАч] ── с встроенной защитой
    │
   [U1: MCP1700T-3002E] ── LDO 3.0 В
    └──► VDD_3V0 → E73
```

## Почему это вынесено в эксперименты

- сильно зависит от реального освещения;
- требует более сложной механики корпуса;
- добавляет деградацию LiPo как отдельный фактор риска;
- усложняет BOM и сборку по сравнению с вариантом на первичной батарее.
