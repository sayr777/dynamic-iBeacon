# Первичная настройка JDY-23

## Назначение этапа

Первичная настройка `JDY-23` выполняется один раз на производстве до пайки батареи. Она переводит модуль в режим `iBeacon` с минимальным рекламным интервалом. Все дальнейшие изменения `Major`, `Minor` и `MAC` выполняет хост-контроллер `STM32L031` в процессе работы изделия.

## Чем отличается от статической версии

В проекте `ble-tag-jdy23` `Major/Minor` устанавливались один раз и навсегда. Здесь первичная настройка задаёт только режим работы. Параметры `Major/Minor/MAC` хост обновляет сразу после первого запуска от батареи.

## Как настраивать

Первичная настройка выполняется от внешнего питания `3.3 В`:

- питание от стабильного источника `3.3 В`;
- `USB-UART TTL 3.3 В`;
- каждая команда отправляется с окончанием `\r\n` (`CRLF`);
- батарея и хост `STM32L031` в этот момент ещё не подключены.

Подтверждённый ответ тестового образца:

```
+VER:JDY-23-V1.2
```

## Базовый профиль

Для этого проекта принят профиль:

- `iBeacon`;
- рекламный интервал `100 мс` (самый частый из стабильных);
- LED-индикатор отключён;
- `Major/Minor` — начальные заглушки `0001/0001` (хост заменит при первом запуске);
- логирование отключено.

Почему `100 мс`, а не `1 с`:

- хост управляет подачей питания на `JDY-23` сам;
- `JDY-23` включается на `~80 мс` и должен успеть передать хотя бы один пакет;
- при интервале `100 мс` первый пакет уходит через `≤ 100 мс` после запуска модуля;
- при интервале `1 с` первый пакет может не успеть уйти за отведённое окно.

## Порядок работ

1. Подать внешнее питание `3.3 В` напрямую на `JDY-23`.
2. Проверить ответ `AT+VER\r\n`.
3. Сбросить профиль `AT+DEFAULT\r\n`.
4. Записать профиль из файла [`jdy23-ibeacon-profile.txt`](jdy23-ibeacon-profile.txt).
5. Перезагрузить модуль `AT+RST\r\n`.
6. Убедиться, что реклама видна в BLE-сканере.
7. Снять питание с `JDY-23`.
8. Подключить хост `STM32L031` и подать питание от батареи.
9. При первом запуске хост установит актуальные `Major/Minor/MAC`.

## Правило отправки команд

- каждая `AT`-команда отправляется с окончанием `\r\n`;
- терминал должен быть настроен на `CRLF`;
- бодрейт: `9600 baud`, `8N1`.

## Что не трогать

- мощность передачи `AT+POWR`;
- `AT+IBSING` (калибровка RSSI);
- параметры, не относящиеся к базовому маячному режиму.
