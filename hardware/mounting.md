# Механическая укладка в корпус

```text
+----------------------------------------------+
| [плата с MCP1700 + YJ-16013]   [антенна →]   |
|                                  keepout     |
|                                              |
|   [ER14505H-LD батарея вдоль длинной оси]    |
|                                              |
+----------------------------------------------+
```

## Обязательные правила

- антенна [YJ-16013](../specs/YJ-16013-datasheet.pdf) направлена к пластиковой стенке корпуса;
- свободный зазор перед антенной `≥ 10 мм`;
- батарея не располагается перед антенной;
- провода батареи прокладываются в стороне от RF-зоны;
- тест-точки `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` доступны до заливки.

## Порядок механической сборки

1. Разместить плату у края корпуса антенной к пластиковой стенке.
2. Уложить батарею вдоль длинной стороны корпуса.
3. Проложить провода вдоль стороны, противоположной антенне.
4. Проверить доступ к тест-точкам и устойчивость фиксации.
5. Подать питание, проверить BLE-рекламу и ток.
6. Залить мягким компаундом только неантенную часть.
7. Выполнить финальный BLE-тест.

## Тестовые точки для обслуживания

- `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` — для прошивки и диагностики;
- при необходимости отдельная контрольная точка `VDD_3V0` для измерения питания.
