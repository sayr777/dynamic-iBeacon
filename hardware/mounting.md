# Механическая укладка в корпус

## Принципы размещения

Те же, что в `ble-tag-jdy23`, с учётом дополнительного узла хоста.

```
+--------------------------------------------------+
|                                                  |
|  [STM32L031 + MOSFET + обвязка]    [JDY-23]     |
|                                    [антенна →→]  |
|                                    keepout 15 мм |
|  [батарея ER14505H-LD]                           |
|                                                  |
+--------------------------------------------------+
```

## Обязательные правила

- **Антенна `JDY-23` — к пластиковой стенке корпуса**, зазор `≥ 10 мм`.
- Батарея — параллельно длинной оси корпуса, **не перед антенной**.
- Хост `STM32L031` и MOSFET — на противоположном от антенны конце.
- Провода UART (`PA9/PA10 ↔ JDY-23`) — как можно короче, не через RF-зону.
- Провода питания — вдали от антенны.
- Заливка компаундом — только неантенная зона.

## Порядок механической сборки

1. Разместить `STM32L031` + обвязку на вспомогательной плате или навесным монтажом.
2. Разместить `JDY-23` рядом, антенной к стенке корпуса.
3. Соединить UART проводниками минимальной длины.
4. Соединить питание (MOSFET, конденсатор, диод).
5. Проверить, что тестовые точки `SWD` доступны снаружи (до заливки).
6. Уложить батарею.
7. Проверить BLE-видимость и управление питанием `JDY-23` до заливки.
8. Залить мягким компаундом неантенную зону.
9. Финальный тест.

## Тестовые точки для обслуживания

Рекомендуется вывести минимальный разъём или тестовые точки:

- `SWDIO`, `SWCLK`, `GND` — для обновления прошивки в поле;
- `UART TX/RX` — для обновления `unix_time` при обслуживании;
- `VCC_MAIN` — для измерения напряжения батареи.
