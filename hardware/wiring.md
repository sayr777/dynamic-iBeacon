# Подключение компонентов

Платформа: **`YJ-16013` (nRF52832)**.

## Распиновка YJ-16013 (используемые пины)

| Пин модуля | GPIO nRF52832 | Функция |
|---|---|---|
| `VCC` | — | питание `VDD_3V0` (3.0 В от MCP1700) |
| `GND` | — | общий минус |
| `SWDIO` | выделенный SWD | программирование SWD |
| `SWDCLK` | выделенный SWD | программирование SWD |
| `RESET` | выделенный RESET | аппаратный сброс (активный низкий) |

Других внешних подключений нет — `nRF52832` полностью автономен.  
Все BLE и AES операции выполняются внутри SoC.

## Схема соединений

```
ER14505H-LD (+) ──► D1(BAT54) ──► CBUF(+) ──► MCP1700 IN ──► VDD_3V0 ──► YJ-16013 VCC
ER14505H-LD (−) ────────────────────────────────────────────────────────── GND
                                               MCP1700 GND ──────────────── GND

TP_SWDIO  ──► YJ-16013 SWDIO
TP_SWDCLK ──► YJ-16013 SWDCLK
TP_RESET  ──► YJ-16013 RESET
TP_GND    ──► GND
TP_VDD    ──► VDD_3V0
```

## Подключение программатора (J-LINK / nRF52 DK / ST-LINK V2)

```
Программатор        Тестовые точки → YJ-16013
────────────        ─────────────────────────
SWDIO    ──────────► TP_SWDIO  → SWDIO
SWDCLK   ──────────► TP_SWDCLK → SWDCLK
RESET    ──────────► TP_RESET  → RESET   ← обязателен для nRF52832
GND      ──────────► TP_GND
3.3V     ──────────► TP_VDD    (питание только от программатора, батарея не подключена)
```

**Важно**: при прошивке через `nrfjprog` RESET-пин используется автоматически.  
Без соединения RESET возможны проблемы при программировании защищённого APPROTECT.

## Что проверить при монтаже

- `D1 (BAT54)` — полярность: анод к батарее `+`, катод к `CBUF`;
- `CBUF` максимально близко к `+` батарейного вывода (после диода);
- `MCP1700` вход и выход с конденсаторами `1 µF` вплотную к корпусу LDO;
- `100 нФ` вплотную к `VCC` пину `YJ-16013`;
- антенна `YJ-16013` направлена к пластиковой стенке корпуса, зазор ≥ 10 мм;
- тест-точки `SWD + RESET` доступны для pogo-pin jig до заливки компаундом.
