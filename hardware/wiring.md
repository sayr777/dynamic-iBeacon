# Подключение компонентов

## Распиновка STM32L010F4P6 (TSSOP-20)

| Пин | Номер | Функция | Назначение |
|---|---:|---|---|
| `PA9` | 17 | USART1 TX | TX → JDY-23 RXD (AT-команды раз в 5 мин) |
| `PA10` | 18 | USART1 RX | RX ← JDY-23 TXD |
| `PA13` | 19 | SWDIO | Программирование SWD |
| `PA14` | 20 | SWCLK | Программирование SWD |
| `VDD` | 1 | Питание | VCC_MAIN через 100 нФ |
| `GND` | 4 | Земля | Общий минус |

P-MOS GPIO не требуется.

## Схема соединений

```
VCC_MAIN ─────────────────── VDD STM32L010 (пин 1)
          │
         100 нФ
          │
         GND (пин 4)

VCC_MAIN ─────────────────── JDY-23 VCC
          │
         100 нФ
          │
         GND

PA9  (пин 17) ─────────────── JDY-23 RXD
PA10 (пин 18) ─────────────── JDY-23 TXD

JDY-23 GND ─────────────────── GND

PA13 (пин 19) ── TP_SWDIO ─── ST-LINK SWDIO
PA14 (пин 20) ── TP_SWCLK ─── ST-LINK SWCLK
```

## Цепь питания

```
ER14505H-LD (+)
    │
   [D1: Schottky] ──► VCC_MAIN
                          │
                    [CBUF 47...100 µF]
                          │
                         GND
ER14505H-LD (-) ────── GND
```

## Что проверить при монтаже

- провода UART (`PA9/PA10 ↔ JDY-23`) — максимально короткие;
- `CBUF` ближе к батарейному входу;
- `100 нФ` максимально близко к пину `VDD` `STM32L010` и к `VCC` `JDY-23`;
- антенна `JDY-23` не перекрыта проводами и батареей.

## Подключение для первичной настройки JDY-23

`STM32L010` не подключается. `JDY-23` питается напрямую от `3.3 В` внешнего источника:

```
Внешний источник 3.3 В ─── JDY-23 VCC
USB-UART адаптер TX ──────── JDY-23 RXD
USB-UART адаптер RX ──────── JDY-23 TXD
GND ──────────────────────── JDY-23 GND
```
