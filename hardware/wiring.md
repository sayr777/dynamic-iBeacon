# Подключение компонентов

## Распиновка E73-2G4M04S1A (используемые пины)

| Пин модуля | GPIO nRF52810 | Функция |
|---|---|---|
| `VCC` | — | питание `VDD_3V0` |
| `GND` | — | общий минус |
| `SWDIO` | `P0.18` | программирование SWD |
| `SWCLK` | `P0.19` | программирование SWD |
| `nRESET` | `P0.21` | сброс (тест-поинт) |

Других внешних подключений нет — `nRF52810` полностью автономен.

## Схема соединений

```
ER14505H-LD (+) ──► CBUF (+) ──► MCP1700 IN ──► VDD_3V0 ──► E73 VCC
ER14505H-LD (−) ──────────────────────────────────────────── GND
                                 MCP1700 GND ──────────────── GND

TP_SWDIO ──► E73 SWDIO
TP_SWCLK ──► E73 SWCLK
TP_GND   ──► GND
TP_VDD   ──► VDD_3V0
```

## Подключение программатора ST-LINK V2

```
ST-LINK V2          E73 / тестовые точки
─────────           ────────────────────
SWDIO    ──────────► TP_SWDIO
SWCLK    ──────────► TP_SWCLK
GND      ──────────► TP_GND
3.3V     ──────────► TP_VDD   (только от ST-LINK, батарея не подключена)
```

## Что проверить при монтаже

- `CBUF` максимально близко к `+` батарейного вывода;
- `MCP1700` вход и выход с конденсаторами `1 µF` вплотную к корпусу LDO;
- `100 нФ` вплотную к `VCC` пину `E73`;
- антенна `E73` направлена к пластиковой стенке корпуса, зазор ≥ 10 мм;
- тест-поинты `SWD` доступны для pogo-pin jig до заливки компаундом.
