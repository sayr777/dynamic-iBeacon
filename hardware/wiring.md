# Подключение компонентов

## Распиновка STM32L031F6P6 (TSSOP-20)

Используемые пины:

| Пин | Номер | Функция | Назначение |
|---|---:|---|---|
| `PA0` | 6 | GPIO output | Управление питанием JDY-23 (затвор P-MOS) |
| `PA9` | 17 | USART1 TX | TX → JDY-23 RXD |
| `PA10` | 18 | USART1 RX | RX ← JDY-23 TXD |
| `PA13` | 19 | SWDIO | Программирование SWD |
| `PA14` | 20 | SWCLK | Программирование SWD |
| `VDD` | 1 | Питание | VCC_MAIN через 100 нФ |
| `GND` | 4 | Земля | Общий минус |

## Схема соединений

```
VCC_MAIN ─────────────────────────────── VDD (пин 1)
          │
         100 нФ
          │
         GND (пин 4)

VCC_MAIN ─── R1(10k) ─── PA0(пин 6) ─── G(затвор Q1 P-MOS)
VCC_MAIN ────────────────────────────── S(исток Q1)
                                         D(сток Q1) ──── JDY-23 VCC

PA9(пин 17) ─────────────────────────── JDY-23 RXD
PA10(пин 18) ────────────────────────── JDY-23 TXD

JDY-23 GND ─────────────────────────── GND

PA13(пин 19) ───── TP_SWDIO ────────── ST-LINK SWDIO
PA14(пин 20) ───── TP_SWCLK ────────── ST-LINK SWCLK
```

## Питание

```
ER14505H-LD (+)
    │
   [D1: Schottky] ──► VCC_MAIN
                          │
                    [CBUF 47...100 µF]
                          │
                         GND
ER14505H-LD (-) ─────── GND
```

## Что проверить при монтаже

- минимальная длина проводов между батареей и входом питания;
- `CBUF` размещать физически ближе к входу питания, чем к модулям;
- `100 нФ` максимально близко к пину `VDD` `STM32L031`;
- провод `GND` батареи и провод `GND` схемы иметь минимальное сопротивление;
- затвор MOSFET подтянут к `VCC_MAIN` через `R1` (защита при старте);
- `JDY-23` антенна не перекрыта батареей и проводами.

## Подключение на этапе первичной настройки JDY-23

Для первичной AT-настройки (`configuration/README.md`) `STM32L031` не подключается. `JDY-23` питается напрямую от `3.3 В` внешнего источника:

```
Внешний источник 3.3 В ─── JDY-23 VCC
USB-UART адаптер TX ──────── JDY-23 RXD
USB-UART адаптер RX ──────── JDY-23 TXD
GND ──────────────────────── JDY-23 GND
```
