# Аппаратная часть

В этом проекте аппаратная часть включает два основных активных компонента: хост-контроллер `STM32L031F6P6` и BLE-модуль `JDY-23`.

## Минимальный состав

- `STM32L031F6P6` в корпусе `TSSOP-20`
- `JDY-23` (готовый модуль или готовая плата)
- `P-channel MOSFET` малой мощности (ключ питания `JDY-23`)
- `10 кОм` резистор (подтяжка затвора MOSFET)
- батарея `ER14505H-LD` с выводами под пайку
- диод Шоттки (защита по питанию)
- конденсатор `47...100 µF` (буфер питания)
- `100 нФ` (развязка питания `STM32L031`)

## Главные инженерные требования

- `STM32L031` питается напрямую от аккумуляторного напряжения через диод Шоттки (1.65...3.6 В — допустимо);
- `JDY-23` питается через MOSFET-ключ, управляемый GPIO хоста;
- при выключенном MOSFET ток через `JDY-23` равен нулю;
- антенная зона `JDY-23` остаётся свободной от компаунда, металла и проводников;
- предусмотреть тестовые точки `SWDIO`, `SWCLK`, `GND`, `VCC` для прошивки `STM32L031`.
