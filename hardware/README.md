# Аппаратная часть

Платформа: **[YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) (nRF52832)**.

## Минимальный состав

| Компонент | Назначение |
|---|---|
| [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) (nRF52832) | BLE SoC, 512KB Flash / 64KB RAM, встроенная PCB-антенна |
| `MCP1700T-3002E/TT` | LDO `3.0 В`, `Iq ~1 µА` |
| `BAT54` (SOD-323) | диод Шотки: защита от переполюсовки |
| `1 µF X7R` × 2 | вход и выход LDO |
| `100 нФ X7R` × 2 | развязка питания модуля |
| `220 µF low-leakage` | буферный конденсатор (пик TX) |
| батарея `ER14505H-LD` | питание, с выводами под пайку |
| тест-точки `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` | прошивка и отладка |

## Ключевые требования

- антенная зона [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) свободна от компаунда, металла и батареи (зазор ≥ 10 мм);
- батарея только с выводами под пайку, без держателя;
- `BAT54` соблюдать полярность: анод → батарея `+`;
- `CBUF` максимально близко к батарейному входу;
- тестовые точки `SWDIO + SWDCLK + RESET` доступны до заливки компаундом.
