# Электрическая схема

Схема идентична [`ble-tag-e73/hardware/schematic.md`](../../ble-tag-e73/hardware/schematic.md).

## Цепь питания

```
ER14505H-LD (+3.6 В)
    │
   [CBUF: 100...220 µF low-leakage]
    │
   [U1: MCP1700T-3002E/TT]
    │  IN ── C_IN (1 µF) ── GND
    │  OUT── C_OUT(1 µF) ── GND  ──► VDD_3V0 (3.0 В)
    │
   GND
         VDD_3V0 ──[C2: 100 нФ]── GND
         VDD_3V0 ──────────────── E73 VCC

ER14505H-LD (−) ─────────────── GND
```

## Программирование (SWD)

```
ST-LINK V2 / J-LINK:
    SWDIO ──► E73 SWDIO (P0.18 на nRF52810)
    SWCLK ──► E73 SWCLK (P0.19 на nRF52810)
    GND   ──► GND
    3.3 В ──► VDD_3V0 (только от программатора, без батареи)
```

Обязательные тестовые точки: `TP_SWDIO`, `TP_SWCLK`, `TP_GND`, `TP_VDD`.

## Сетевые имена

| Net | Описание |
|---|---|
| `VBAT` | напряжение батареи до LDO (3.3–3.6 В) |
| `VDD_3V0` | стабилизированное питание 3.0 В → E73 |
| `SWDIO` | SWD данные |
| `SWCLK` | SWD тактирование |
| `GND` | общий минус |

## Почему MCP1700 (не прямое подключение)

- `nRF52810` рассчитан на `1.7...3.6 В`; свежая батарея даёт `3.6 В` — граница;
- `MCP1700` обеспечивает стабильные `3.0 В` и `Iq ~1 µА` — не влияет на ресурс;
- та же логика, что в `ble-tag-e73`.

## Критичные правила

- `CBUF` — физически ближе к батарейному входу LDO;
- `C_IN`, `C_OUT`, `C2` — максимально близко к соответствующим пинам;
- антенная зона `E73` (PCB-антенна со стороны модуля) — без металла, без компаунда, зазор ≥ 10 мм.
