# Электрическая схема

## Цепь питания

```
ER14505H-LD (+3.6 В)
    │
    ├──[D1: Schottky BAT48]──► VCC_MAIN (≈3.5 В)
    │
    │   VCC_MAIN
    │     │
    │     ├──[CBUF: 47...100 µF low-leakage]──┐
    │     │                                    │
    │     ├──[C1: 100 нФ]──┐                  │
    │     │                │                  │
    │   STM32L031          │                GND
    │   VDD ◄──────────────┘
    │
    │   VCC_MAIN
    │     │
    │     R1 (10 кОм) ──► G (затвор Q1)
    │                      │
    │              Q1: P-channel MOSFET (например, AO3407)
    │              S (исток) ──► VCC_MAIN
    │              D (сток)  ──► JDY-23 VCC
    │              G         ◄── PA0 (STM32L031)
    │
    └─── GND ──────────────────────────────── GND
```

**Логика управления MOSFET:**

- `PA0 = LOW` (STM32L031 вытягивает к GND) → Q1 открыт → `JDY-23 VCC = VCC_MAIN` (включён)
- `PA0 = HIGH` (STM32L031 вытягивает к VDD) → Q1 закрыт → `JDY-23 VCC = 0` (выключен)
- Резистор `R1 10 кОм` подтягивает затвор к `VCC_MAIN` при сбросе `STM32L031` (защита от случайного включения `JDY-23`)

## Интерфейс UART (STM32L031 ↔ JDY-23)

```
STM32L031 PA9  (USART1 TX) ──────► JDY-23 RXD
STM32L031 PA10 (USART1 RX) ◄────── JDY-23 TXD
Уровни: 3.3 В (совместимы)
Baud: 9600, 8N1
```

## Программирование STM32L031

```
Программатор ST-LINK V2
    SWDIO  ──► PA13 (STM32L031)
    SWCLK  ──► PA14 (STM32L031)
    GND    ──► GND
    3.3 В  ──► VCC_MAIN (только от программатора, без батареи)
```

Тестовые точки обязательны:
- `TP_SWDIO`, `TP_SWCLK`, `TP_GND`, `TP_VCC`

## Рекомендуемый P-channel MOSFET

| Параметр | Требование | AO3407 |
|---|---|---|
| Тип | P-channel | ✓ |
| `Vgs(th)` | `< 2.5 В` | `-0.4...-1.8 В` |
| `Id` | `> 50 мА` | `4 А` |
| `Rdson` | `< 1 Ом` | `40 мОм` |
| Корпус | `SOT-23` | `SOT-23` |
| Цена | дёшево | `$0.05` |

## Сетевые имена

| Net | Описание |
|---|---|
| `VCC_MAIN` | основная шина питания после диода (≈3.5 В) |
| `VCC_JDY` | питание `JDY-23` (управляется MOSFET) |
| `SWDIO` | SWD данные |
| `SWCLK` | SWD тактирование |
| `GND` | общий минус |
| `JDY_RXD` | UART к JDY-23 |
| `JDY_TXD` | UART от JDY-23 |
| `JDY_PWR_CTRL` | PA0, управление MOSFET |
