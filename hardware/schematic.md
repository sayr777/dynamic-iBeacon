# Электрическая схема

Платформа: **[YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) (nRF52832)**.

## Цепь питания

```
ER14505H-LD (+3.6 В)
    │
   [D1: BAT54] ── SOD-323 — защита от переполюсовки
    │
   [CBUF: 220 µF low-leakage] ── буфер пикового тока TX
    │
   [U1: MCP1700T-3002E/TT]
    │  IN  ── C_IN  (1 µF X7R) ── GND
    │  OUT ── C_OUT (1 µF X7R) ── GND  ──► VDD_3V0 (3.0 В)
    │  GND ────────────────────────────── GND
    │
   VDD_3V0 ──[C2: 100 нФ X7R]── GND
   VDD_3V0 ───────────────────── YJ-16013 VCC
   GND     ───────────────────── YJ-16013 GND
```

## Программирование (SWD + RESET)

```
J-LINK / nRF52 DK / ST-LINK V2:
    SWDIO  ──► TP_SWDIO  → YJ-16013 SWDIO
    SWDCLK ──► TP_SWDCLK → YJ-16013 SWDCLK
    RESET  ──► TP_RESET  → YJ-16013 RESET  ← пин RESET есть на nRF52832
    GND    ──► TP_GND
    3.3V   ──► TP_VDD    (только от программатора, без батареи)
```

`nRF52832` имеет выделенный пин `RESET` (в отличие от nRF52810).  
Это упрощает SWD-отладку: можно сбросить чип без отключения питания.

## Сетевые имена

| Net | Описание |
|---|---|
| `VBAT` | напряжение батареи до диода и LDO (3.3–3.6 В) |
| `VDD_3V0` | стабилизированное питание 3.0 В → [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) |
| `SWDIO` | SWD данные |
| `SWDCLK` | SWD тактирование |
| `RESET` | сброс nRF52832 (активный низкий) |
| `GND` | общий минус |

## Почему MCP1700 (не прямое подключение)

- `nRF52832` рассчитан на `1.7...3.6 В`; свежая батарея даёт `3.6 В` — граница;
- `MCP1700` обеспечивает стабильные `3.0 В` и `Iq ~1 µА` — не влияет на ресурс;
- диод `BAT54` (Vf ~0.3 В) снижает напряжение с `3.6 В` до `~3.3 В` на входе LDO.

## Почему CBUF 220 µF

- `nRF52832` TX-пик: `5.3 мА × ~1 мс` = `5.3 µКл`;
- `MCP1700` не успевает подать пиковый ток при малом Iq;
- `220 µF × 0.1 В` допустимого провала = `22 µКл` — запас `×4`.
- Тип: `low-leakage` (Nichicon UCC / Panasonic FR): утечка `< 0.5 µА`.

## Критичные правила разводки

- `CBUF` — физически ближе к батарейному входу LDO;
- `C_IN`, `C_OUT`, `C2` — максимально близко к соответствующим пинам;
- антенная зона [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) — без металла, без компаунда, зазор ≥ 10 мм от корпуса;
- тест-точки `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` — доступны до заливки компаундом.
