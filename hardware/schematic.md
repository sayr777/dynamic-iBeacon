# Электрическая схема

## Цепь питания

```
ER14505H-LD (+3.6 В)
    │
   [D1: Schottky BAT48]──► VCC_MAIN (≈3.5 В)
    │
    │   VCC_MAIN
    │     ├──[CBUF: 47...100 µF low-leakage]──┐
    │     │                                    │
    │     ├──[C1: 100 нФ]──┐                  │
    │     │                │                  │
    │   STM32L010          │                GND
    │   VDD ◄──────────────┘
    │
    │   VCC_MAIN ──────────────────────────── JDY-23 VCC
    │             [C2: 100 нФ]               (питание постоянное)
    │                  │
    │                 GND
    │
    └─── GND ─────────────────────────────── GND
```

**P-MOS ключ не используется.**  
`JDY-23` подключён к `VCC_MAIN` напрямую и работает непрерывно.

## Интерфейс UART (STM32L010 ↔ JDY-23)

```
STM32L010 PA9  (USART1 TX) ──────► JDY-23 RXD
STM32L010 PA10 (USART1 RX) ◄────── JDY-23 TXD
Уровни: 3.3 В (совместимы)
Baud: 9600, 8N1
```

UART активен только ~600 мс раз в 5 мин при обновлении параметров.  
В `Stop mode` TX-пин `STM32L010` находится в состоянии `Hi-Z` или `HIGH` (IDLE),  
что не мешает `JDY-23` (он ждёт данных, не выдаёт команды).

## Программирование STM32L010

```
Программатор ST-LINK V2
    SWDIO ──► PA13 (STM32L010)
    SWCLK ──► PA14 (STM32L010)
    GND   ──► GND
    3.3 В ──► VCC_MAIN (только от программатора, без батареи)
```

Тестовые точки обязательны: `TP_SWDIO`, `TP_SWCLK`, `TP_GND`, `TP_VCC`.

## Сетевые имена

| Net | Описание |
|---|---|
| `VCC_MAIN` | основная шина питания после диода (≈3.5 В) |
| `SWDIO` | SWD данные |
| `SWCLK` | SWD тактирование |
| `GND` | общий минус |
| `JDY_RXD` | UART TX → JDY-23 |
| `JDY_TXD` | UART RX ← JDY-23 |

## Почему нет P-MOS

Включение/выключение `JDY-23` через GPIO-ключ каждые 2 с создаёт средний ток `600 µА` (из-за `~15 мА × 80 мс / 2 с`). Это сокращает ресурс батареи до `~6 месяцев`.

При постоянном питании `JDY-23` работает на своём внутреннем цикле сна/рекламы с потреблением `≈ 17 µА`, что даёт расчётный ресурс `>10 лет`.

## Критичные правила

- `CBUF` размещать физически ближе к батарейному входу;
- `C1, C2` максимально близко к питанию соответствующих модулей;
- зону антенны `JDY-23` не загружать посторонними цепями;
- `JDY_RXD` и `JDY_TXD` не прокладывать через антенную зону.
