# Спецификация модуля YJ-16013

## Назначение

Этот документ собирает в одном месте ключевую информацию по модулю `YJ-16013`, который используется в проекте как базовая аппаратная платформа для динамической BLE-метки.

## Идентификация модуля

| Параметр | Значение |
|---|---|
| Модель модуля | `YJ-16013` |
| Производитель / линейка | Holyiot / YJ series |
| Базовый SoC | `nRF52832` |
| Архитектура | ARM Cortex-M4F |
| Основное применение | BLE 5.0 модуль с PCB-антенной |

## Ключевые параметры

| Параметр | Значение |
|---|---|
| Flash | `512 KB` |
| RAM | `64 KB` |
| Питание | `1.7–3.6 В` |
| Радио | Bluetooth Low Energy |
| Антенна | встроенная PCB-антенна |
| Интерфейс прошивки | `SWD` |

## Что важно для проекта

- модуль подходит для прошивки через `SWD`;
- позволяет использовать `nRF5 SDK 17.1.x` и `SoftDevice S112`;
- поддерживает нашу схему глубокого сна и коротких BLE advertising events;
- встроенная антенна требует keepout-зону в корпусе и на плате.
