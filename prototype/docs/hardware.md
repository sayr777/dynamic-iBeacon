# Аппаратная часть прототипа

## ProMicro NRF52840 v1940 — распиновка

```
              USB-C
         ┌────┴────┐
    P0.11 ┤ 1    18 ├ P1.09
     P0.9 ┤ 2    17 ├ P0.31
    P0.29 ┤ 3    16 ├ P0.30
    P0.28 ┤ 4    15 ├ P0.02
    P0.20 ┤ 5    14 ├ P0.03
    P0.17 ┤ 6    13 ├ P0.04
    P0.15 ┤ 7    12 ├ P0.05
    P0.18 ┤ 8    11 ├ P0.09
       5V ┤ 9    10 ├ 3.3V
      GND ┘         └ GND
             [LED]
         [JST-PH 2-pin]
              BATT+
              BATT-
```

| Пин платы | GPIO nRF52840 | Функция в прототипе |
|---|---|---|
| 1 | `P0.11` | — (свободный) |
| 2 | `P0.09` | **LED синий** (статус обновления параметров) |
| 3 | `P0.29` | — |
| 4 | `P0.28` | — |
| 5 | `P0.20` | — |
| 6 | `P0.17` | — |
| 7 | `P0.15` | — |
| 8 | `P0.18` | — |
| 9 | `5V` | питание от USB |
| 10 | `3.3V` | выход LDO (нагрузка ≤ 25 мА) |
| 11 | `P0.09` | LED (дублируется в таблице) |
| 18 | `P1.09` | — |

Для нашего iBeacon-прототипа **никакие внешние пины не нужны** —  
всё выполняется внутри nRF52840 (BLE, AES, RTC).

## Схема питания

### Вариант A — питание от USB-C (для отладки)

```
USB-C (5 В)
    │
   [nRF52840 внутренний DC-DC/LDO]
    │
   3.3 В → nRF52840 ядро, BLE радио, AES
    │
   [LTH7R] ← заряжает LiPo если подключён
```

### Вариант B — питание от LiPo без USB (автономный тест)

```
LiPo 3.7 В (JST-PH 2-pin)
    │
   [LTH7R] → ~20 µА standby при полном заряде
    │
   [nRF52840 внутренний DCDC]
    │
   3.3 В / 1.8 В → ядро, BLE, RTC
```

## Компоненты на плате

| Компонент | Назначение | Параметры |
|---|---|---|
| `nRF52840` | SoC: CPU + BLE + AES + RTC + USB | 1 MB Flash, 256 KB RAM |
| `LTH7R` | LiPo зарядник | 100 мА заряд, **~20 µА standby** |
| PCB-trace антенна | BLE радио | встроенная, без u.FL |
| LED синий | статус (прошивка + debug) | `P0.09`, активный низкий |
| USB-C | прошивка (UF2) + питание | нативный USB nRF52840 |
| JST-PH 2-pin | LiPo аккумулятор | 3.7 В, полярность: красный=`+` |

## SWD тест-пады (обратная сторона платы)

```
┌──────────────────────┐
│  [SWDIO]  [SWDCLK]   │  ← припаять тонкий провод
│  [RST]    [GND]      │
│  [VCC]               │
└──────────────────────┘
```

Для прошивки через SWD (nRF5 SDK или west flash --runner jlink):

```
J-LINK     ProMicro (SWD-пады)
───────    ─────────────────────
SWDIO  ──► SWDIO
SWDCLK ──► SWDCLK
GND    ──► GND
3.3V   ──► VCC  (питание от J-LINK, LiPo не подключён!)
```

Для UF2-прошивки через USB — SWD не нужен.

## Подключение LiPo аккумулятора

```
LiPo (3.7 В):
  красный провод (+) ──► JST BATT+
  чёрный провод (−)  ──► JST BATT−
```

**Внимание**: полярность JST-PH на разных клонах платы может отличаться.  
Проверить мультиметром до подключения! Перепутать полярность — сжечь плату.

Рекомендуемый аккумулятор для прототипа: **LiPo 100–500 мАч с JST-PH 2-pin**.  
При 25 µА потреблении:
- 100 мАч → ~100 дней автономной работы
- 500 мАч → ~500 дней (~1.4 года)

## Вход в режим загрузчика UF2

Для перепрошивки через USB без J-LINK:

1. Замкнуть `RST` на `GND` **два раза быстро** (двойной клик)
2. Плата появится как USB-диск `NRF52BOOT` или `FTHR840BOOT`
3. Скопировать `.uf2` файл на диск
4. Плата перезагрузится автоматически
