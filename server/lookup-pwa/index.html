Как пользоваться

1. Откройте nRF Connect → Scan → тапните на метке → Manufacturer Data

2. Скопируйте UUID, Major, Minor

3. Или возьмите из serial-лога прошивки: UUID=... Major=0xXXXX Minor=0xXXXX

⚠️ UUID, Major, Minor и MAC — все динамические, меняются каждый слот

⚠️ KEY должен совпадать с ключом в прошивке метки

Алгоритм (схема v2)
для TagID в [tMin..tMax]:
  для Slot в [sMin..sMax]:
    b0 = TagID[2] ‖ Slot[4] ‖ 0x00 ‖ 0(9)   ← variant=0
    b1 = TagID[2] ‖ Slot[4] ‖ 0x01 ‖ 0(9)   ← variant=1
    uuid_cand = AES-128-ECB(KEY, b0)
    adv_cand  = AES-128-ECB(KEY, b1)
    если uuid_cand==UUID И adv_cand[0:2]==Major И adv_cand[2:4]==Minor:
      TagID найден!
      MAC = adv_cand[4]|0xC0 ‖ adv_cand[5:10]
