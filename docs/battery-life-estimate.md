# Оценка срока жизни батареи

## Важное: почему нельзя включать JDY-23 каждые 2 с

Первоначально в проекте рассматривалась схема с P-MOS ключом, который включает `JDY-23` на `80 мс` каждые `2 с`. Расчёт показывает, что это **неприемлемо**:

```
I_avg(JDY-23) = 15 мА × 80 мс / 2000 мс = 600 µА
Ресурс батареи: 2700 мАч / 600 µА = 4500 ч ≈ 6 месяцев ❌
```

Принятое решение: `JDY-23` работает непрерывно на внутреннем цикле рекламы `2 с`. Средний ток модуля в этом режиме — `≈ 17 µА`.

## Исходные данные (принятая архитектура)

- батарея `FANSO ER14505H-LD`: `2700 мАч` номинально, `≈2160 мАч` консервативно доступная
- `JDY-23` непрерывная реклама, интервал `2 с`: `≈ 17 µА` (ориентир из документации, нужно подтвердить измерением)
- `STM32L010F4P6` в `Stop mode` с `LSE 32768 Гц`: `0.29 µА`
- `STM32L010` активный во время AT-обновления (раз в `5 мин`, `~600 мс`): `~2 мА`
- Утечки схемы (диод, конденсатор): `< 0.5 µА`

## Расчёт среднего тока

### JDY-23 (непрерывный)

```
I_JDY = 17 µА  (постоянно)
```

### STM32L010 Stop mode

```
I_stop = 0.29 µА × (300 с − 0.6 с) / 300 с ≈ 0.289 µА
```

### STM32L010 активный (AT-обновление каждые 5 мин)

```
I_active = 2 мА × 0.6 с / 300 с = 4.0 µА
```

### Утечки

```
I_leak = 0.4 µА
```

### Итого

```
I_avg = 17 + 0.289 + 4.0 + 0.4 ≈ 22 µА
```

## Ресурс батареи

### Только нагрузочный ток

```
Life_load = 2160 мАч / 22 µА / 8760 ч/год ≈ 11.2 лет
```

### С учётом саморазряда Li-SOCl₂

Саморазряд `FANSO ER14505H-LD`: `~1%` в год = `27 мАч/год`.

```
Расход в год = 22 µА × 8760 ч + 27 мАч = 193 + 27 = 220 мАч/год
Ресурс        = 2160 мАч / 220 мАч/год ≈ 9.8 лет
```

### Безопасный план

При использовании `2160 мАч` (80% от номинала):

| Ресурс | Значение |
|---|---|
| Расчётный | `≈ 9.8 лет` |
| С учётом деградации и температуры | `≈ 5–7 лет` |
| **Проектный целевой срок** | **3 года** |
| **Запас** | **×2.5 от проектного** |

Целевой срок `3 года` обеспечивается с запасом `>2×`.

## Чувствительность к параметрам JDY-23

Самый значимый параметр — средний ток `JDY-23`. Его нужно измерить на реальной плате.

| Средний ток JDY-23 | Полный средний | Ресурс (с саморазрядом) |
|---:|---:|---:|
| `10 µА` (оптимистично) | `15 µА` | `13 лет` |
| `17 µА` (базово) | `22 µА` | `9.8 лет` |
| `30 µА` (пессимистично) | `35 µА` | `6.5 лет` |
| `50 µА` (очень плохой модуль) | `55 µА` | `4.2 лет` |

**Даже при самом пессимистичном сценарии** (`50 µА` от `JDY-23`) цель `3 года` выполняется.

## Сравнение вариантов

| Вариант | Средний ток | Ресурс |
|---|---:|---:|
| `ble-tag-jdy23` (статичный, `1 с` интервал) | `≈ 50 µА` | `≈ 5 лет` |
| **ble-tag-jdy23-dynamic (данный проект)** | **`≈ 22 µА`** | **`≈ 9.8 лет`** |
| Проект с nRF52810 (без JDY-23) | `≈ 4–5 µА` | саморазряд `>20 лет` |
| ❌ Неправильная схема (MCU power cycling 2 с) | `≈ 600 µА` | `≈ 6 мес.` |

## Рекомендации для производства

```
Целевой ток JDY-23 в режиме реклама 2 с:  < 30 µА
Обязательная проверка: измерить ток на реальной плате до запуска партии
Целевой суммарный средний ток изделия:    < 50 µА
Инженерный план:                           3 года при ≤ 55 µА
```
