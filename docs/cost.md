## Схема питания — каждый компонент и зачем

```
ER14505H-LD  (+3.6 В)
     │
    [D1: BAT54]          Vf ≈ 0.30 В при 1 мА → защита от переполюсовки
     │                   Падение: 3.6В − 0.3В = 3.3В на входе LDO
     │
    [CBUF: 220 µF]       Запасает: 220 µF × 0.1 В = 22 µКл
     │  low-leakage      Нужно TX: 5.3 мА × 1 мс = 5.3 µКл → запас ×4
     │  утечка < 0.5 µА  Тип: Nichicon UCC / Panasonic FR
     │
    [U1: MCP1700T-3002E] LDO 3.0 В
     ├─ C_IN  (1 µF) ── GND   стабильность входа LDO
     └─ C_OUT (1 µF) ── GND ──► VDD_3V0 (3.0 В)
                                    │
                                   [C2: 100 нФ] ── GND  (высокочастотный байпас)
                                   [C3: 100 нФ] ── GND  (у второго VCC-пина YJ-16013)
                                    │
                                   [YJ-16013 VCC]
```

**Почему MCP1700, а не прямое подключение:**
- `nRF52832` max `3.6 В` — батарея в начале жизни даёт `3.6 В` (граница);
- LDO стабилизирует `3.0 В` на всё время жизни батареи;
- `Iq = 1 µА` — не влияет на ресурс (≈ 20% от тока deep sleep).

**Почему `low-leakage` CBUF, а не обычный:**
- Обычный электролит: утечка `5–20 µА` → убьёт весь бюджет (`5.2 µА` общий ток);
- `Nichicon UCC` / `Panasonic FR`: утечка `< 0.5 µА` — допустимо.

## Ресурс батареи

| Режим | Средний ток | Ресурс расчётный | Ресурс практический |
|---|---:|---:|---:|
| Без ночного режима | 5.2 µА | ~30 лет | **~20 лет** |
| С ночным режимом (23:00–06:00) | 4.4 µА | ~35 лет | **~24 года** |

Цель проекта — **3 года**. Запас: **×7** без ночного режима, **×8** с ним.

Ограничивающий фактор — **саморазряд** батареи (~1%/год = 27 мАч/год),  
а не нагрузочный ток (~45 мАч/год при 5.2 µА).

## Стоимость и результат

| Показатель | Значение |
|---|---:|
| Полная стоимость изделия | **$6.95** |
| Средний ток | **~5.2 µА** |
| Практический ресурс батареи | **~15–20 лет** |
| Целевой срок проекта | **3 года** |

Текущее решение немного дороже простых BLE-маяков, но выигрывает за счёт:

- динамической смены эфирных параметров;
- однокристальной архитектуры на `nRF52832`;
- одного маршрута производственной прошивки через `SWD`;
- большого запаса по автономности.

## Что проверить перед закупкой

| Компонент | Что проверить |
|---|---|
| [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) | убедиться что чип `nRF52832`, а не другая ревизия |
| `MCP1700T-3002E/TT` | именно `3002` (3.0 В), не `3302` (3.3 В) |
| `220 µF CBUF` | тип `low-leakage`: Nichicon UCC, Panasonic FR/FM, Rubycon ZLH |
| `BAT54` | любой аналог: BAT60A, RB520S, Vf ≤ 0.35 В при 1 мА |
| `ER14505H-LD` | исполнение **с выводами под пайку**, не в держателе |
| PCB | вынести тест-точки `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` |
| Компаунд | **не заливать** антенную зону [YJ-16013](https://device.report/shenzhen-holyiot-technology/nrf52832) |
