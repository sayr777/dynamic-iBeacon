## Базовый корпус

Текущее решение ориентировано на корпус `AK-W-70-4` с размещением:

- основной платы с [YJ-16013](../specs/YJ-16013-datasheet.pdf);
- батареи `ER14505H-LD`;
- свободной антенной зоны без металла и компаунда.

## Схема укладки

```text
+----------------------------------------------+
| [плата с MCP1700 + YJ-16013]   [антенна →]   |
|                                  keepout     |
|                                              |
| [батарея ER14505H-LD вдоль длинной стороны]  |
|                                              |
+----------------------------------------------+
```

## Критичные правила

- антенна [YJ-16013](../specs/YJ-16013-datasheet.pdf) направлена к пластиковой стенке корпуса;
- перед антенной нужен зазор не менее `10 мм`;
- батарея не должна перекрывать антенную зону;
- провода батареи укладываются вдали от RF-зоны;
- тест-точки `SWDIO`, `SWDCLK`, `RESET`, `GND`, `VDD` должны быть доступны до герметизации.

## Правила заливки компаундом

- использовать мягкий полиуретановый компаунд или нейтральный силикон;
- допускается заливка батарейных выводов и силовой части;
- **не заливать** антенную зону [YJ-16013](../specs/YJ-16013-datasheet.pdf) и пространство перед ней;
- жёсткий эпоксид нежелателен из-за механических напряжений и деградации RF.

## Порядок сборки

1. Проверить плату после прошивки и BLE-теста.
2. Уложить плату в корпус с правильной ориентацией антенны.
3. Припаять батарею и уложить провода вне RF-зоны.
4. Проверить запуск и рекламу от батареи.
5. Залить только неантенную зону.
6. Выдержать компаунд по инструкции производителя.
7. Выполнить финальный BLE-тест после сборки.

## Что проверить после заливки

- дальность связи не просела относительно образца без заливки;
- компаунд не попал в антенную область;
- в корпусе нет смещения батареи;
- ток потребления остался в ожидаемом диапазоне.
