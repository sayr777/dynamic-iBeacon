# Партнёрский сервис локального позиционирования

## Состав

2. Продукт и стратегия

4. Безопасность и compliance
5. Публичный API
6. Интеграция партнёров
7. Sandbox
8. Биллинг и квоты

## Использование

- для внешней интеграции стартовой точкой являются `04-api`, `05-partner-integration`, `06-sandbox`;
- для операционного запуска и SLA используются `08-operations` и `07-billing`;
- для коммерческого и юридического согласования используются `09-commercial`, `10-finance`, `11-legal-and-public-sector`;
- для пилотов, презентаций и ответов партнёрам используется `12-enablement`.
