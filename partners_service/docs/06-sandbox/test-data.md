# Test Data

Тестовый набор данных включает выделенный AES-ключ для Sandbox, тестовый UUID, ограниченный набор объектов и несколько контрольных сценариев для успешных и неуспешных вызовов API.

## Требования к набору

- идентификаторы объектов отделены от production-реестра;
- координаты и значения сигналов воспроизводимы между прогонами;
- ожидаемый ответ для каждого контрольного payload заранее известен;
- обновление тестового набора версионируется вместе с sandbox-конфигурацией.
