# Sandbox Overview

Sandbox — отдельный контур для интеграции партнёров, проверки схем запросов и демонстрации API без влияния на продуктивный поток. Он использует тестовые ключи, тестовый справочник и детерминированные ответы на контрольные payload.

## Состав контура

- sandbox gateway с отдельными ключами доступа;
- ingestion mock или упрощённый входной слой с полной валидацией схемы;
- decryption mock на тестовом наборе ключей;
- registry emulator с тестовыми объектами;
- отдельные метрики и журналы, не смешиваемые с production.
