# Procurement 44-FZ

Материал используется для подготовки закупочного пакета на предоставление доступа к сервису локального позиционирования в государственном контуре.

## Основные части

- предмет закупки;
- функциональные требования к сервису;
- показатели SLA;
- порядок сдачи-приёмки;
- требования к защите технических данных;
- ответственность сторон и порядок расторжения.

## Практический результат

Документ позволяет быстро собрать закупочную документацию и согласовать её с техническим и юридическим блоком без повторного описания архитектуры и API с нуля.

## Минимальный комплект для закупки

- описание предмета закупки;
- техническое задание с функциональными требованиями;
- показатели SLA;
- методика приёмки;
- требования к размещению и защите данных;
- форма отчётности по оказанным услугам.
