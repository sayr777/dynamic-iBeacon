# ROI Model

Финансовая модель оценивает самоокупаемость сервиса за счёт коммерческого доступа к уже развёрнутой инфраструктуре.

## Ключевые драйверы

- количество активных партнёров;
- распределение по тарифам;
- фактический объём запросов и overage;
- стоимость поддержки, инфраструктуры и биллингового контура;
- скорость выхода на новый регион или новый класс дорожных объектов.

## Основные метрики

- выручка по сценариям;
- точка безубыточности;
- ROI;
- NPV;
- IRR.
